## 📊 Project Overview
This project analyzes customer support tickets to identify common issues, evaluate service quality, and improve customer satisfaction.

### Objectives
- Clean and standardize ticket data  
- Analyze issue patterns and priorities  
- Extract meaningful insights  
- Provide actionable recommendations  

---

##  Data Source
- Simulated dataset (manually created for learning)
- Total tickets analyzed: **21**

###  Fields:
- Ticket Number  
- Customer Name  
- Issue Description  
- Priority Level  

---

##  Tools & Technologies
- **Language:** Python  
- **Environment:** Google Colab / Jupyter Notebook  
- **Libraries:** Built-in Python (no external libraries used)  

---

##  Data Cleaning & Preparation
- Removed punctuation (`.,!?-`)  
- Converted text to lowercase  
- Removed extra spaces  
- Replaced slang words:
  - `ok → okay`  
  - `pls → please`  
  - `thx → thanks`  
  - `asap → as soon as possible`  
- Standardized text for consistent analysis  

---

##  Exploratory Data Analysis (EDA)

###  Performed Analysis:
- Ticket-wise formatted display  
- Priority distribution  
- Keyword-based issue analysis  
- Longest issue identification  
- Unique word extraction  

---

##  Key Insights
- High-priority tickets are the highest → critical issues exist  
- Internet-related problems are most common  
- Slow response is a frequent complaint  
- Feedback is mixed (positive + negative)  
- Service quality is inconsistent  
- Repeated issues indicate poor resolution efficiency  
- Some issues are complex and detailed  
- Early signs of dissatisfaction are visible  

---

##  Recommendations
- Fix core technical issues (internet, login)  
- Improve response time  
- Ensure consistent service quality  
- Implement SLA (Service Level Agreements)  
- Categorize tickets for faster handling  
- Create FAQ / self-service support  
- Monitor negative keywords (slow, poor)  
- Track customer sentiment  
- Improve first-time resolution  
- Continuously monitor performance  

---

##  Conclusion
- The system is working but needs improvement  
- Main issues are internet problems and slow response  
- Service quality is inconsistent  
- Some issues are repeated and unresolved  
- Improving key areas will increase customer satisfaction
